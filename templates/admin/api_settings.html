Cách lấy OpenAI API Key

  1. Đăng nhập vào tài khoản OpenAI của bạn tại platform.openai.com
  2. Truy cập vào mục API Keys trong phần quản lý tài khoản
  3. Tạo API key mới và sao chép vào ô nhập liệu ở trên
  4. Nhấn "Lưu API Key" để cập nhật

Lưu ý quan trọng

  • API key có tính phí, hãy đảm bảo bạn hiểu rõ cách tính phí của OpenAI
  • Không chia sẻ API key với người khác
  • Nếu API key bị lộ, hãy vô hiệu hóa ngay và tạo key mới
  • Kiểm tra trạng thái API key thường xuyên để đảm bảo hệ thống hoạt động ổn định
